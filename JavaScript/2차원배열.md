### 문법
타입 배열이름[열의길이][행의길이];

### 예시1) arr[2][3]
```
arr[0][0] arr[0][1] arr[0][2]  
arr[1][0] arr[1][1] arr[1][2]
```

### 예시2) 
```
let animals = [   
    ["고양이", "강아지", "곰돌이", "코끼리"],  
    [1, 2, 3, 4],  
    ["야옹", "멍멍", "곰돌", "뿌우"]   
];
```

console.table(animals)를 사용하면 콘솔창에서 2차원 배열을 쉽게 알아볼 수 있다.
```
index   0      1     2      3
  0    고양이  강아지  곰돌이  코끼리
  1     1      2     3      4
  2    야옹    멍멍   곰돌    뿌우
```

```
// animals[0][0]
// animals[0][1]
// animals[0][2]
// animals[0][3]

// => 동물이름구하기
for(let i=0; i<animals[0].length; i++) {
    animals[0][i]
}
```
